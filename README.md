# Pocket Arcade 🎮

オフライン対応の **PWA ミニゲーム集**。slither 系の「**スネークアリーナ**」と、
ヴァンパイアサバイバー風の「**ルミナ・サバイバー**」を収録。スマホ・PC どちらでも遊べます。

## 🌑 ルミナ・サバイバー（ヴァンパイアサバイバー風・オリジナル世界）

> 永い夜「常闇（とこやみ）」に呑まれた世界。あなたは最後の灯火 **ルミナ**。
> 押し寄せる影「ヨル」を退け、夜明けまで生き延びろ。

- **動かすだけ**: 武器は自動で発動。指でドラッグ（バーチャルスティック）／PC は WASD・矢印キー。
- **倒すと光片（経験値）がドロップ** → 一定量でレベルアップ → **3つの力から1つ選択**。
- **10人のキャラクター**: 性能（HP・攻撃・速度・連射・範囲・会心など）が異なり、初期武器も別。
  3人は最初から、残りは **アンロック条件**（撃破数・到達Lv・生存時間・ステージクリア等）で解放。
- **12種の光武器＋12種の進化武器**: ルクス弾・光輪・聖環・新星・連鎖雷・流星・氷晶・聖鞭・
  重力・ブーメラン・光槍・パルス。**武器を最大Lv＋対応パッシブを最大**にすると **進化**（例: ルクス弾→プリズム・ランス）。
- **14種のパッシブ**: 攻撃力・連射・移動速度・最大HP・再生・取得範囲・防御・経験値・範囲・弾速・
  持続・投射数・会心・幸運。
- **6つのステージ**: 黄昏の草原 / 常闇の樹海 / 凍てつく霊峰 / 灰の都 / 血染めの月 / 夜明け前。
  ステージごとに色調・敵編成・難度が変化し、前を **クリアすると次が解放**。各ステージ専用ボスつき
  （ボスは倒しやすく再調整）。
- **8種類のBGM**: メニュー＋ステージ6種＋ボス戦、すべて WebAudio で合成（ボス出現で曲が切替）。
- **リッチな演出**: ベイク済みグロウスプライト＋加算合成のネオン表現、パーティクル、
  画面シェイク、ダメージ数字、ビネット、漂う光の塵。スコアは **生存時間**（ステージ別の自己ベスト保存）。

## 🐍 スネークアリーナ

イライラしないように調整した slither 系バトル。

## あそびかた

ローカルサーバーで開くだけ（Service Worker / PWA のため `file://` ではなく http が必要）:

```bash
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

スマホでは「ホーム画面に追加」するとアプリとして起動し、**オフラインでも**遊べます。

### 操作
- **なぞる**（デフォルト）／**スワイプ**: 設定で切替。指の方向にヘビが進みます。
- **ブースト**: 右下のボタン（PC は Space）。少しずつ大きさを消費して加速。
- PC はマウスで方向、Space でブースト。

## こだわりポイント（リクエスト対応）
- **倒したエサに群がらない**: 倒した相手のエサは数秒間「倒した本人専用」に予約され、
  他のヘビは寄ってこない（`CLAIM_MS`）。AI のエサ探索もばらけるよう調整。
- **常にライバルが出る**: プレイヤーが大きくなると、近い〜少し上のサイズの敵が必ず
  湧くように個体数とサイズを動的調整（無理ゲーにならない上限つき）。
- **スマホ対応 / スワイプ操作**、**効果音**（WebAudio で生成・ファイル不要）。
- **気持ちよさ**: 食べる連鎖コンボ、パーティクル、画面シェイク、なめらかカメラ＆ズーム。
- **ライフ＆補充**: ライフは死亡で減り、3分で1回復。さらに⭐を5個集めると即時+1。
- **その他**: スコア＆ベスト記録、リアルタイム順位ボード、ミニマップ、
  パワーアップ（🛡シールド / 🧲マグネット / ⚡スピード / ⭐スター）。

## 技術
- 素の HTML / CSS / JS（フレームワークなし）。Canvas 2D で描画。
- PWA: `manifest.webmanifest` + `sw.js`（キャッシュファースト）。
- 効果音・BGM はすべてコードで合成（外部音声ファイル無し → 完全オフライン）。
- データ（ライフ・ベスト・設定）は `localStorage` に端末ローカル保存。

## 構成
```
index.html              アプリ本体（オープニング＋ゲーム画面）
styles/main.css         UI スタイル
js/storage.js           ライフ回復・スコア（ゲーム別ベスト）・設定の永続化
js/audio.js             効果音 / BGM 合成エンジン
js/snake.js             スネークアリーナ本体（描画・物理・AI）
js/lumina.js            ルミナ・サバイバー エンジン（描画・敵AI・武器archetype・進化・ステージ）
js/lumina-weapons.js    武器＆パッシブ定義（データ）
js/lumina-chars.js      キャラクター定義（データ）
js/lumina-stages.js     ステージ＆敵定義（データ）
js/app.js               画面遷移・HUD・各種ボタン配線
manifest.webmanifest    PWA マニフェスト
sw.js                   Service Worker（オフライン）
icons/                  アイコン（生成物）
tools/gen_icons.py      アイコン生成スクリプト
tools/headless_test.js  スネークのスモークテスト（node）
tools/lumina_test.js    ルミナのスモークテスト（node）
```

### テスト
```bash
node tools/headless_test.js   # スネーク: 4600 フレームで例外・主要不変条件を確認
node tools/lumina_test.js     # ルミナ: 敵湧き・撃破・経験値・レベルアップ・被弾死亡を確認
```
